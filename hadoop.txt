带操作习惯

vi /etc/selinux/config 	编辑selinux配置文件
setenforce 0 	暂时关闭selinux
/etc/init.d/iptables status 	查看防火墙
/etc/init.d/iptables start 	启动防火墙
/etc/init.d/iptables stop 	关闭防火墙
/etc/init.d/iptables restart 	重启防火墙
chkconfig iptables off 	开机禁用防火墙
chkconfig iptables on 	开机启用防火墙
vi /etc/ssh/sshd_config	编辑关闭selinux
vim /etc/ssh/sshd_config 	编辑关闭selinux
/etc/init.d/sshd restart 	重启ssh


1.关闭selinux(国外的类似iptables)
vi /etc/selinux/config
setenforce 0 	暂时关闭selinux
SELINUX=disabled 	在selinux配置文件内添加关闭
2.关闭防火墙
/etc/init.d/iptables stop 	关闭防火墙
chkconfig iptables off 	开机禁用防火墙
3.配置无密钥登陆
ssh-keygen -t dsa 	生成密钥
cat id_dsa.pub >> authorized_keys 	生成密钥后追加到authorized_keys
scp -r  authorized_keys  root@192.168.70.136:~/.ssh 从本地远程复制密钥传到指定计算机
4.ssh比较慢的时候
vi /etc/ssh/sshd_config
找到#UseDNS yes 修改为UseDNS no
然后保存退出service sshd restart重启服务
登陆延迟等待
vim /etc/resolv.conf 清空内容

vi /etc/inittab
